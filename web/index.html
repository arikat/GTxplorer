GT Xplorer

An interactive portal to visualize and mine evolutionary data for glycosyltransferases with a GT-A type fold.

From the ESBG lab,UGA

GTXplorer provides an interactive platform for users to access, navigate, download and visualize the evolutionary information from primary protein sequences of the GT-A fold families. There are 2 main ways to navigate.

Tree view

Click on clade and family names for a pop up card with all the information.

Alignment view

Stack alignments of Clades, Families and Subfamilies for cross-comparison of conserved pattern positions.

4 types of files are available for download from GTXplorer:
  • FASTA formatted full lenght GT-A sequences (from NCBInr and UniProt databases).
  • FASTA formatted sequences for the GT-A domains only (for both NCBInr and UniProt).
  • Sequence information in a tabular format. This tab delimited file has the most detailed information about all the GT-A sequences.
  • Domain organization information for each of the 53 families.

To download these files, look for these icons: (for NCBInr data) and (for UniProt data)
